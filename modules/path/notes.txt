## Key Methods

### path.join()
Joins path segments with proper separators:
```javascript
path.join('/foo', 'bar', 'baz'); // /foo/bar/baz
```

### path.resolve()
Resolves to absolute path:
```javascript
path.resolve('file.txt'); // /current/dir/file.txt
```

### path.parse()
Breaks path into components:
```javascript
path.parse('/home/user/file.txt');
// { root: '/', dir: '/home/user', base: 'file.txt', ext: '.txt', name: 'file' }
```

## Why Important?
- Cross-platform compatibility (Windows vs Unix paths)
- Avoid hardcoded separators
- Safe path construction
- Working with file uploads and static assets
