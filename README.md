## Quick Access

### Learn Node.js
1. Read `node-theory.txt` (5 min revision)
2. Open `node-notes.js` (commented code examples)
3. Practice in `modules/` folder

### Learn Express.js
1. Read `express-theory.txt` (5 min revision)
2. Open `express-notes.js` (commented code examples)
3. Practice in `express/` folder

### Learn MongoDB
1. Read `mongodb-theory.txt` (5 min revision)
2. Open `mongodb-notes.js` (commented code examples)
3. Practice in `mongodb-examples/` folder

