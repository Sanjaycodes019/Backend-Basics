This folder demonstrates file upload handling using Multer middleware in Express.js.

## Key Concepts

### Multer Configuration
```javascript
const upload = multer({ 
  dest: 'uploads/'  // Destination folder
});
```

### Upload Methods
- `upload.single(fieldname)` - Single file upload
- `upload.array(fieldname, maxCount)` - Multiple files with same field name
- `upload.fields(fields)` - Multiple files with different field names

### File Information (req.file)
- `originalname` - Original file name
- `mimetype` - File MIME type
- `size` - File size in bytes
- `filename` - Stored file name
- `path` - Storage path

## How to Test
1. Run: `node 01multer.js`
2. Use Postman or curl to send POST request to `/upload`
3. Include file in form-data with key matching the configured field name
